## Features
- eslint
- prettier
- babel
- stylus

## Requirements
You need to have <b>node.js</b> installed. 

## Setup
Install dependencies
```sh
$ npm install
```

## Development
Run the local webpack-dev-server with hotreload and autocompile on [http://localhost:8080/](http://localhost:8080/)
```sh
$ npm run dev
```

## Deployment
Build the current application
```sh
$ npm run build
```
